# VoiceInput - 语音输入法

基于 Web Speech API 的智能语音输入法，支持实时语音转文字。

## 功能

- 🎤 按住说话，实时识别
- ✏️ 识别结果可编辑
- 📋 一键复制
- 🎯 智能标点（AI辅助）

## 技术栈

- 前端：HTML5 + Web Speech API
- 后端：Java Spring Boot
- AI：硅基流动 Qwen2.5-7B

## 运行

```bash
cd frontend
# 直接打开 index.html 即可
```

## 项目结构

```
├── frontend/     # 前端界面
└── backend/      # 后端服务
```
